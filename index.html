Code for China

Code for China 是一个程序员小组，关注并实践公共服务的数据化工作。
我们为公共数据提供 HTTP 接口服务，并维护这些开源项目的代码。

国家数据

获取国家数据库 2013 年 GDP 数据

$ curl -x GET \
  http://api.codeforchina.org/national/gdp/2013

法律

获取民事诉讼法第一章第一条

$ curl -x GET \
  http://api.codeforchina.org/law/civil/c1/1

国家标准

获取国家标准行政区划代码

$ curl -x GET \
  http://api.codeforchina.org/gb/gb2260? \
  code = 420822

$ ==>
  湖北省 荆门市 沙洋县

公共交通

获取杭州市公共交通巴士数据

$ curl -x GET \
  http://api.codeforchina.org/traffic/hangzhou? \
  type = bus
